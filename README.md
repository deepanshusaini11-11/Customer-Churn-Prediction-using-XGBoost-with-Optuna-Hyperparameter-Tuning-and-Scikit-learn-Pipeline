# Customer Churn Prediction 🚀

This project builds a robust machine learning pipeline to predict customer churn using XGBoost and Optuna for hyperparameter optimization.

## 🔥 Key Features
- End-to-end ML pipeline using Scikit-learn
- Automated preprocessing with ColumnTransformer
- OneHotEncoding for categorical variables
- Handling missing values and datetime features
- Hyperparameter tuning using Optuna
- Cross-validation for reliable performance

## 🛠️ Tech Stack
- Python
- Scikit-learn
- XGBoost
- Optuna
- Pandas & NumPy

## 📊 Model Performance
- Optimized using 5-fold cross-validation
- Metric: Accuracy

## 🚀 Workflow
1. Data Cleaning & Preprocessing
2. Feature Engineering
3. Pipeline Creation
4. Model Training (XGBoost)
5. Hyperparameter Tuning (Optuna)
6. Evaluation
